\section{微積分: テイラー展開と数値積分}
\subsection{\(e^x\)、sin \(x\)、cos \(x\)を
それぞれマクローリン展開しなさい}

\[
 e^x = \sum^{\infty}_{n=1}\frac{x^n}{n!}
\]

\[
 \sin x = \sum^{\infty}_{n=1} (-1)^n \frac{x^{2n-1}}{(2n-1)!}
\]

\[
 \cos x = \sum^{\infty}_{n=0} (-1)^n \frac{x^{2n}}{(2n)!}
\]

\subsection{ (1)の結果を用いてオイラーの公式
\(e^{i\theta } = {\rm cos} \theta + i\ {\rm sin} \theta \)を証明しなさい。}

\[
 e^{ix} = 1 + \frac{1}{1!}i\theta + \frac{1}{2!}(i\theta)^2 + \cdots
\]

\[
 e^{ix} = \{ 1 - \frac{1}{2!}\theta^2 + \cdots \} + 
 i\{ \theta - \frac{1}{3!}\theta^3 + \cdots \}
\]

\[
 e^{ix} = \cos \theta + i \sin \theta
\]

\subsection{\(y = f(x)\)をテイラー展開し、オイラー法\(y_{n+1}=y_{n} + h
f^{\prime}(x_n)\)が1次の数値積分法(=誤差項が2次)であることを示しなさい。}

y=f(x)をaの回りでテイラー展開すると
\[
 y = f(a) + f'(a)(x-a) + \frac{f''(a)}{2!} (x-a)^2 + \cdots
\]
オイラー法でh=x-aなので、
\[
y_{n+1} = y_n + f'(x_n)(x-a) 
\]
よって2次以降は誤差となる。
